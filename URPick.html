Robot Pick and Place

While working as an applications engineer for an automation equipment distributor, a customer needed a demonstration of a Cognex Smart Camera being used as part of a Pick & Place Robot setup. I created this basic demonstration using a Universal Robot 6 axis arm and communicated with the Smart Camera using Ethernet IP.

The Camera setup includes a bar light and bright pink background in order to get good contrast. I could then use a Pattern Finding tool on the Cognex camera to identify the location and orientation of the part. I then transformed the pixel coordinate system of the camera into real world coordinates by using a printed calibration grid.

I then transformed the carmeras coordinate system into the robots coordinate system using a transformation matrix. Once the coordinate systems were identical, I was able to pass the location of the part to the robot using the Ethernet IP registers. For the purpose of the demo, I had the robot hand me the object where I could then replace it.
